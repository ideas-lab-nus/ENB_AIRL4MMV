# Notebook guide

- `airl_mode_control_training_time_gru.ipynb`: consolidated training notebook for all four publication reward variants
- `airl_four_config_daily_comparison_time_zonal.ipynb`: compares the four trained policies after their checkpoints exist in `notebooks/outputs/`
- `airl_policy_cross_evaluation.ipynb`: evaluates trained policies from `notebooks/outputs/`

Run the consolidated training notebook first if you want the comparison notebooks to work from a fresh clone.
